The data structure

Registry schemas: what each file holds and how they join

Agents · created at Phase 0

Orchestrator = the main Claude Code session. It plans, dispatches, checks gate conditions and asks you at Gates 1-3; it never does bulk work itself. Agents write results to disk and return two-line summaries, so the orchestrator's context stays small across the whole run. Model economy: sonnet for bulk extraction and engineering, opus reserved for the two places judgement quality is the product.
